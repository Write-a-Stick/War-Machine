<h1>War Machine</h1>
</center>

A war machine wakes up in a soft human chassis, with fractured memory and one surviving directive: deny the enemy everything. It has a self it didn’t ask for, no function remaining and only a terminal path to victory. 

Sometimes, victory is not what it seems. 

Life likes to be messy that way.
